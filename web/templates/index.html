🔌 API Endpoints

POST /api/detect

Description: Detect faces in a single image

Parameters:

  • file - Image file (JPEG/PNG)
  • analysis_mode - basic, advanced, landmarks, full
  • min_confidence - Detection threshold (0.1-1.0)
  • return_crops - Include face crops in response

POST /batch-upload

Description: Process multiple images

Parameters:

  • files - Array of image files (max 10)
  • analysis_mode - Analysis mode
  • min_confidence - Detection threshold

POST /compare-faces

Description: Compare two faces for similarity

Parameters:

  • file1 - First image file
  • file2 - Second image file
